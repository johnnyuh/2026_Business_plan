  ## DOCUMENT NAMING CONVENTIONS

    - ALL CAPS

    - UNDERSCORES

    - HYPHENS

  ### Core Naming Formula

    TYPE_PROJECT/CLIENT_DETAIL_CONTRACT_DATE_SUFFIX.ext

  #### TYPE

    - LIST_

    - BUY_

    - MKT_

    - MKT-RPT_

    - POST_ 

    - SOC_ 

    - YT_

    - IG_

    - TT_

    - FB_

    - SUBS_

    - KPI_

    - BP_2026_

    - WEEK_

    - QTR_

    - LASER_


  #### PROJECT/CLIENT

    - FARM-1

    - SURNAME


  #### DETAIL

    - "DAYS-IN-MLS_"
  
  #### CONTRACT

    - example: LC50-6-24

  #### DATE

    - Format: YYYYMMDD

    - The date the file was created.

  #### SUFFIX

    - _FINAL Deliverable or approved file

    - _SIGNED Executed documents

    - _REVISED Corrected version

    - _EXPORT File prepared for external distribution
